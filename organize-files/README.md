# Organize Files Skill

## Overview

**Organize Files** helps turn messy files into a clean folder structure with clear names, logical grouping, and a downloadable archive.

## What This Skill Does

Use this skill when you need to clean up a set of files, rename them consistently, group them by project or category, or package the result for sharing.

It can help with:

- sorting files into a meaningful hierarchy
- renaming files with a consistent scheme
- grouping by topic, client, type, or date
- producing a final zip archive with the organized structure

## Sample Prompts For Testing

### Sort a Mixed Folder

```text
Organize the files below into a clean folder structure.
Group them by type and topic, and rename unclear file names so they are easy to understand.
```

### Client Asset Cleanup

```text
Reorganize these client files into a professional folder hierarchy.
Use clear names and make the final result easy to hand off to another person.
```

### Research Archive

```text
Take these research documents and organize them by theme, date, and usefulness.
Rename the files so I can quickly scan what each one contains.
```

### Downloads Folder Cleanup

```text
Clean up this downloads folder.
Sort the files, rename ambiguous ones, and package the organized result into a zip archive.
```

## Typical Outputs

- organized folder taxonomy
- renamed files with a clear naming scheme
- summary of organizational decisions
- downloadable zip archive
