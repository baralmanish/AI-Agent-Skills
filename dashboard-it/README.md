# Dashboard-It Sample Prompts

Use these prompts when you want to invoke the `dashboard-it` skill with clearer intent.

These prompts are intended to produce a self-contained HTML + JavaScript dashboard that can be previewed, downloaded, and opened offline.

## Quick Prompts

### 1. Conversation Summary Dashboard

```text
Dashboard it.
Turn our current chat thread into a clean interactive dashboard.
Show the key decisions, open questions, priorities, and next actions.
Make the most important takeaways visible first.
```

### 2. Project Status Dashboard

```text
Create a dashboard for the project update below.
Show progress, blockers, milestones, owners, due dates, and next steps.
Use filters for status and owner, and highlight what is at risk.
```

### 3. Roadmap Dashboard

```text
Build a product roadmap dashboard from the plan below.
Group work by quarter, priority, and team.
Include a timeline view, dependency highlights, and a section for critical launches.
```

## Detailed Prompts

### 4. KPI and Metrics Dashboard

```text
Create a dashboard showing the metrics below.
Make the top KPIs prominent, show trends over time, and flag any anomalies.
Add controls for filtering by region, category, and time range.
```

### 5. Team Workload Dashboard

```text
Dashboard the team workload data below.
Show capacity, assigned work, overdue items, and who is overloaded.
Make it easy to compare team members and spot bottlenecks quickly.
```

### 6. Learning Plan Dashboard

```text
Create a dashboard for the learning plan below.
Show modules, completion progress, deadlines, study focus areas, and recommended next lessons.
Make it feel motivating and easy to scan.
```

### 7. Personal Productivity Dashboard

```text
Build a personal productivity dashboard from the notes below.
Show goals, habits, completed tasks, pending work, and weekly focus areas.
Include a simple progress summary and a view of what needs attention today.
```

### 8. Meeting Notes Dashboard

```text
Turn the meeting notes below into a dashboard.
Show decisions made, action items, owners, deadlines, and unresolved questions.
Make follow-up items easy to filter and review.
```

### 9. Workflow Dashboard

```text
Create a dashboard for the workflow below.
Map the steps, show inputs and outputs, highlight bottlenecks, and identify where delays happen.
Use clear visual grouping and drill-down details for each stage.
```

### 10. Time Zone Dashboard

```text
Create a dashboard showing the current time across all time zones.
Include live updates, search, region filters, UTC offsets, and a mobile-friendly layout.
Make the design bold and modern, and let me open it offline in a browser.
```

### 11. Decision Dashboard

```text
Dashboard the decision framework below.
Show options, tradeoffs, risks, benefits, cost, effort, and recommendation.
Make comparison easy and highlight the strongest option.
```

### 12. Financial Overview Dashboard

```text
Build a dashboard for the budget and spending data below.
Show total spend, category breakdown, monthly trends, savings opportunities, and unusual spikes.
Add filters and make the most important financial insights obvious.
```
