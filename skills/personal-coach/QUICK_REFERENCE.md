# Personal Coach — Quick Reference

## Trigger Phrases

```
"Coach me"
"What should I work on this week?"
"What pattern do you see in how I work?"
"Give me a weekly experiment"
"What's holding me back?"
"Coach me on [communication / prioritization / leadership / decision-making]"
```

## Output (Always These 5 Things)

1. One hidden strategic insight (observed from conversation — not generic)
2. One weekly change (single behavior, 7 days, <15 min/day)
3. `What | Why | How` action table
4. Simple tracking plan (daily metric + weekly outcome metric)
5. CSV copy-paste block + link to `assets/weekly-coaching-tracker.csv`

## Behavior Change Models Used

| Model                         | When Applied                                                 |
| ----------------------------- | ------------------------------------------------------------ |
| Tiny Habits (BJ Fogg)         | User tried and failed before — reduce friction first         |
| Implementation Intentions     | User has intent but doesn't follow through — add specificity |
| Identity-Based Habits (Clear) | User is shifting professional identity (IC → leader)         |
| If-Then Planning              | Known obstacle exists (busy schedule, energy dips)           |

## Common Coaching Themes

| Theme                     | Signals                                                   |
| ------------------------- | --------------------------------------------------------- |
| Communication & Influence | Ideas not getting traction, hard to be heard              |
| Prioritization & Focus    | Everything urgent, calendar full but progress slow        |
| Visibility & Recognition  | Hard work not being noticed, others promoted first        |
| Decision Quality          | Second-guessing, long deliberation on small choices       |
| Delegation & Leverage     | Doing work that should be delegated, bottlenecked on self |

## Rules

- **One change only** — multiple changes dilute focus
- **Testable** — binary (did/did not) or countable
- **Low friction** — under 15 minutes per day
- **Strengths-first** — identify the hidden opportunity in a strength before the gap
