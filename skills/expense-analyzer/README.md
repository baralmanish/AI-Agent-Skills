# Expense Analyzer Skill

## Overview

**Expense Analyzer** turns credit card bills or transaction exports into a rich spending analysis with dashboards, budgeting insights, savings ideas, and scenario planning.

## What This Skill Does

Provide spending data such as a card statement, transaction list, or categorized expenses. The skill can help you:

- identify spending patterns and outliers
- surface recurring subscriptions and hidden charges
- build a budget view by category
- model what-if savings scenarios
- generate a polished dashboard for review

## Sample Prompts For Testing

### Analyze a Statement

```text
Analyze my credit card statement below.
Show category totals, unusual spending, recurring charges, and the top opportunities to save.
Create a dashboard I can review easily.
```

### Build a Budget Dashboard

```text
Create a budget dashboard from the transaction data below.
Show monthly totals, spending by category, trends over time, and where I am overspending.
```

### Subscription Review

```text
Review the expenses below and find subscriptions, repeated charges, and services I may have forgotten about.
Highlight what looks removable or negotiable.
```

### Savings Scenario

```text
Use the spending data below to model a savings plan.
Show what happens if I reduce dining, shopping, and ride-share spending over the next 6 months.
```

## Typical Outputs

- spending summary by category
- savings recommendations
- subscription analysis
- scenario-based savings projections
- interactive or polished dashboard output
