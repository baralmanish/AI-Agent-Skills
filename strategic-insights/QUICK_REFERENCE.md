# Strategic Insights — Quick Reference

## Trigger Phrases

```
"Give me strategic insights from this conversation"
"What career opportunity am I missing?"
"What pattern do you see in how I work?"
"What's my professional blind spot?"
"Help me think about my growth strategically"
"Strategic insight for [leadership / communication / career transitions]"
```

## Output (Always These 5 Things)

1. One hidden strategic insight (cited from observed conversation patterns — not generic)
2. One weekly change (single behavior, 7 days, low-friction)
3. `What | Why | How` action table
4. Tracking plan (daily indicator + weekly outcome metric)
5. CSV copy-paste block + link to `assets/weekly-insight-tracker.csv`

## Strategic Frameworks Used

| Framework                    | What It Identifies                                           |
| ---------------------------- | ------------------------------------------------------------ |
| Ikigai                       | Alignment between love, skill, need, pay                     |
| Strength Zone (Hendricks)    | Zone of Genius vs. Zone of Excellence trap                   |
| Career Capital (Newport)     | Rare skills, reputation, relationships, credentials          |
| Visibility vs. Impact Matrix | High-impact/low-visibility is most common missed opportunity |
| Contribution Ladder          | Current level (1–5) and what next level looks like           |

## Insight Quality Rules

- Must reference specific observed behavior — not personality labels
- Bad: "You are strategic" / Good: "You consistently reframe problems at a higher level than asked"
- Max 3 sentences for the insight
- One behavior change, always

## Weekly Experiment Examples

| Theme         | Experiment                                            |
| ------------- | ----------------------------------------------------- |
| Visibility    | Weekly 3-bullet progress note to manager every Friday |
| Communication | Speak first in next 3 team meetings                   |
| Focus         | 3 priorities before email, every morning              |
| Relationships | One specific 3-sentence outreach per day              |
| Decisions     | Write decision + one-sentence rationale before acting |
